# Display Picture

Component to present a Picture in several variants.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library?type=design&node-id=1-5&mode=design&t=tXbQWENNIzmFNTVU-0) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/displayPicture)

## Usage

Basic import and additional information on how to use this component.

```js
import { DisplayPicture } from '@banyan_cloud/roots';
```
