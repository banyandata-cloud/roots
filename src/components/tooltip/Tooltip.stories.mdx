## Usage

Basic import and additional information on how to use this component.

```js
import { Tooltip } from '@banyan_cloud/roots';
```

-   The `className` of the Tooltip can be adjusted in the implementaion code to match its flexibility with respect to its container.

## API

**`className`**

Key to change style through external className.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>String</td>
		<td>''</td>
	</tr>
</table>

```jsx
<Tooltip className='chart-area' />
```

**`variant`**

The color variant of the container to select between `light` or `dark` mode.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>String</td>
		<td>"dark"</td>
	</tr>
</table>

```jsx
<Tooltip variant='light' />
```

**`position`**

This key decides the position of Tooltip, among the options `right`, `top`, `bottom` and `left`.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>String</td>
		<td>'always'</td>
	</tr>
</table>

```jsx
<Tooltip position='right' />
```
