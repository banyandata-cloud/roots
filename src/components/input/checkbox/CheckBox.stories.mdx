# CheckBox

The Checkbox component provides a customizable checkbox input with an optional label. It supports controlled and uncontrolled usage, allowing you to manage the checked state through its onChange handler.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library?node-id=151%3A18733&mode=dev) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/input/checkbox)

> The Checkbox component offers a simple and customizable way to include checkbox inputs in your application. It supports both controlled and uncontrolled usage, making it suitable for various scenarios.

## Usage

Basic import and additional information on how to use this component.

```js
import { CheckBox } from '@banyan_cloud/roots';
```

-   The position prop allows you to control the positioning of the checkbox relative to the label.
-   The onChange function is called when the checkbox's state changes. It provides the new checked state as a parameter.

## API

**`disabled`**

Disables the checkbox.

	</tr>

```jsx
<CheckBox disabled={true} />
```

**`label`**

Text label associated with the checkbox.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>

