# Badge

A compact label component used to categorize, highlight, or tag content. Supports three variants — `pill`, `badge`, and `modern` — and three sizes — `sm`, `md`, and `lg`.

> **Naming note**: The `color` variant has been renamed to `badge`. There is no "Color Badge" — there is only **Badge** (variant `badge`). All three variants share an identical layout system; only visual properties (background, border, color, border-radius) differ between them.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/v2/badges)

## Usage

```js
import { Badge } from '@banyan_cloud/roots';
```

### onClose

Renders a close/remove button when provided. The badge layout (padding and gap) is identical to other closable variants of the same size.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>() =&gt; void</td>
		<td>undefined</td>
	</tr>
</table>

```jsx
<Badge size='md' variant='pill' label='Label' onClose={() => handleRemove()} />
```
