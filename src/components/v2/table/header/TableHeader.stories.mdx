# Table Header

Just extends the Base Cell with the property of text overflow ellipsis.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library?type=design&node-id=1-5&mode=design&t=tXbQWENNIzmFNTVU-0) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/table/header)

> Container of Table Header.

## Usage

```js
import { TableHeader } from '@banyan_cloud/roots';
```
