# Link

The **Link** component is a text-based navigational element that supports
different sizes, optional icons, disabled state, and standard anchor behavior.
It is built on top of the `Text` component and styled to match the Banyan design system.

#### Design -

[Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library?type=design&node-id=1-5&mode=design&t=tXbQWENNIzmFNTVU-0)
| Code -
[Github](https://github.com/banyandata-cloud/roots/tree/main/components/link)

---

## Usage

```js
import { Link } from '@banyan_cloud/roots';
```

```jsx
<Link href='https://banyancloud.io'>Visit Banyan</Link>
```

## API

### `children`

Content rendered inside the link.

---

### `href`

URL the link points to.

| TYPE   | DEFAULT |
| ------ | ------- |
| string | â€”       |

```jsx
<Link href='/dashboard'>Go to Dashboard</Link>
```

---

### `target`

Specifies where to open the linked document.

| TYPE                         | DEFAULT |
| ---------------------------- | ------- |
| `_self`, `_blank`, `_parent` | `_self` |

```jsx
<Link href='https://example.com' target='_blank'>
	Open External Link
</Link>
```

---

### `size`

Controls font size and spacing.

| TYPE             | DEFAULT |
| ---------------- | ------- |
| `sm`, `md`, `lg` | `md`    |

```jsx
<Link size='lg'>Large Link</Link>
```

---

### `withIcon`

Shows arrow icon next to text.

| TYPE    | DEFAULT |
| ------- | ------- |
| boolean | false   |

```jsx
<Link withIcon>Link with Icon</Link>
```

---

### `disabled`

Disables interaction and navigation.

| TYPE    | DEFAULT |
| ------- | ------- |
| boolean | false   |

```jsx
<Link disabled>Disabled Link</Link>
```

---

### `className`

Apply custom styles.

| TYPE   | DEFAULT |
| ------ | ------- |
| string | `''`    |

```jsx
<Link className='my-custom-link'>Styled Link</Link>
```

---

### `attrs`

Additional HTML attributes forwarded to `<a>`.

```jsx
<Link attrs={{ 'aria-label': 'custom-link' }}>Accessible Link</Link>
```

---

## Notes

- Disabled links prevent navigation and keyboard focus.
- Icons automatically scale based on `size`.
- Fully keyboard accessible with `:focus-visible`.
