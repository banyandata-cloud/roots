# Checkbox

A fully accessible checkbox component with support for checked, unchecked, and indeterminate states across 4 sizes and 7 visual states.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/v2/checkbox)

## Usage

```js
import { Checkbox } from '@banyan_cloud/roots';
```
