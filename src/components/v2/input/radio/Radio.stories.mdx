# Radio

The Radio component provides a customizable radio input for single-selection options. It supports controlled and uncontrolled usage, and covers all interaction states: Enable, Focus, Read-only, Error, Warning, and Disabled.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library?node-id=151%3A18731&mode=dev) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/input/radio)

> The Radio component provides an accessible and easily customizable way to implement radio inputs in your application. It supports both controlled and uncontrolled usage, with options for positioning, sizing, and feedback states.

## Usage

```js
import { Radio } from '@banyan_cloud/roots';
```

- The `onChange` function fires when the radio state changes, receiving `(event, value)`.
- The `position` prop places the radio icon left or right of the label.
- The `size` prop controls the icon size.
- The `readOnly` prop blocks interaction without the full disabled greying.
- The `error` prop shows a red badge icon with a black message below the radio.
- The `warning` prop shows a yellow badge icon with a black message below the radio.
