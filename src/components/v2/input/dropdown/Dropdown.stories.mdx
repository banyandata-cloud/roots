## Props

| Prop                    | Type                                                                                      | Default                               | Description                                          |
| ----------------------- | ----------------------------------------------------------------------------------------- | ------------------------------------- | ---------------------------------------------------- |
| `label`                 | `string`                                                                                  | -                                     | Label text for the dropdown                          |
| `placeholder`           | `string`                                                                                  | `'Select the option'`                 | Placeholder text when no option is selected          |
| `helperText`            | `string`                                                                                  | `'This is a hint text to help user.'` | Helper text below the dropdown                       |
| `value`                 | `string`                                                                                  | -                                     | Controlled value (single select)                     |
| `defaultValue`          | `string`                                                                                  | -                                     | Default value for uncontrolled usage (single select) |
| `selectedValues`        | `string[]`                                                                                | -                                     | Controlled values (multi-select)                     |
| `defaultSelectedValues` | `string[]`                                                                                | -                                     | Default values for uncontrolled usage (multi-select) |
| `options`               | `DropdownOption[]`                                                                        | `[]`                                  | Array of dropdown options                            |
| `state`                 | `'default' \| 'selected' \| 'focus' \| 'disabled' \| 'error' \| 'warning' \| 'read-only'` | `'default'`                           | Visual state of the dropdown                         |
| `size`                  | `'sm' \| 'md'`                                                                            | `'sm'`                                | Size variant                                         |
| `variant`               | `'simple' \| 'multi-select' \| 'with-icon'`                                               | `'simple'`                            | Dropdown variant                                     |
| `required`              | `boolean`                                                                                 | `true`                                | Whether the field is required                        |
| `disabled`              | `boolean`                                                                                 | `false`                               | Whether the dropdown is disabled                     |
| `readOnly`              | `boolean`                                                                                 | `false`                               | Whether the dropdown is read-only                    |
| `helpIcon`              | `boolean`                                                                                 | `true`                                | Whether to show help icon in label                   |
| `errorIcon`             | `boolean`                                                                                 | `true`                                | Whether to show error icon in error state            |
| `warningIcon`           | `boolean`                                                                                 | `true`                                | Whether to show warning icon in warning state        |
| `icon`                  | `React.ComponentType`                                                                     | -                                     | Leading icon component                               |
| `onChange`              | `(value: string) => void`                                                                 | -                                     | Single select change handler                         |
| `onMultiSelectChange`   | `(values: string[]) => void`                                                              | -                                     | Multi-select change handler                          |
| `onFocus`               | `(e: React.FocusEvent<HTMLElement>) => void`                                              | -                                     | Focus event handler                                  |
| `onBlur`                | `(e: React.FocusEvent<HTMLElement>) => void`                                              | -                                     | Blur event handler                                   |
| `className`             | `string`                                                                                  | -                                     | Additional CSS class                                 |
| `id`                    | `string`                                                                                  | -                                     | HTML id attribute                                    |
| `name`                  | `string`                                                                                  | -                                     | HTML name attribute                                  |
