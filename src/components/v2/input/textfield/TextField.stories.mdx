# TextField

Text input field component with multiple states from the Figma design system.

#### Design - [Figma](https://www.figma.com/design/IfmLQrA0AAZVYwt6vRxqCe/Design-System--26?node-id=1981-3711&m=dev) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/src/components/v2/input/textfield)

## Usage

Basic import and additional information on how to use this component.

```js
import { TextField } from '@banyan_cloud/roots';
```

## API

**`label`**

Label text displayed above the input field.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>string</td>
		<td>undefined</td>
	</tr>
</table>

```jsx
<TextField label='Email Address' />
```

**`placeholder`**

Placeholder text shown when input is empty.

<table style={{ width: '100%' }}>
	<tr>

		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>

</table>


Helper text displayed below the input field.
