# Breadcrumbs

A breadcrumb navigation component for React applications that helps users track their current location within a hierarchical structure. The component supports multiple visual types, separator styles, and dropdown options per crumb.

#### Design - [Figma](https://www.figma.com/design/IfmLQrA0AAZVYwt6vRxqCe/Design-System--26?node-id=1018-697&p=f&m=dev) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/breadcrumbs)

> The Breadcrumbs component renders a navigation bar that displays breadcrumb items. It supports three visual types - `text`, `text-with-line`, and `button` - along with two separator styles - `chevron` and `slash`. Each crumb can have an optional dropdown with selectable options.

## Usage

Basic import and additional information on how to use this component.

```js
import { Breadcrumbs } from '@banyan_cloud/roots';
```

- This Breadcrumbs component provides users with a clear visual representation of their current navigation path.
- It supports `text`, `text-with-line`, and `button` types with `chevron` or `slash` separators.
- Each crumb supports a dropdown with clickable options for contextual navigation.

## API

**`type`**

The visual type of the breadcrumbs.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>'text', 'text-with-line', 'button'</td>
		<td>'text'</td>
	</tr>
</table>

**`separator`**

The separator icon between breadcrumb items.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>'chevron', 'slash'</td>
		<td>'chevron'</td>
	</tr>
</table>

**`crumbs`**

An array of breadcrumb items with properties.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>{`[{ label: string, onClick: func, isDisabled: bool, dropdownOptions: array }]`}</td>
		<td>[]</td>
	</tr>
</table>

**`className`**

Additional CSS class for the breadcrumb container.

	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>string</td>
		<td>''</td>
	</tr>
</table>
