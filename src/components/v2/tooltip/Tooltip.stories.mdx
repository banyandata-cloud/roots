# Tooltip

A highly dynamic layout wrapper for providing dense contextual information on hover or focus. Fully supports collision-aware screen-edge repositioning and a brand new interactive mode.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/src/components/v2/tooltip)

## Usage

Basic import and additional information on how to use this component.

```tsx
import { Tooltip } from '@banyan_cloud/roots';
```

- The `className` of the Tooltip can be adjusted in the implementation code to match its flexibility with respect to its container.
- By passing `interactive={true}`, the tooltip becomes natively clickable before dismissing, allowing rich custom HTML components like links or buttons to be embedded safely inside the floating layer.

## API

**`content`**

The string or interactive React node to display inside the tooltip body.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>ReactNode</td>
		<td>Required</td>
	</tr>
</table>

```jsx
<Tooltip content='Tooltip Info' />
```

**`position`**

This key decides the position of Tooltip, among the options `top`, `bottom`, `left` and `right`.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>String</td>
		<td>'top'</td>
	</tr>
</table>

```jsx
<Tooltip position='right' />
```

**`interactive`**

Whether the tooltip allows pointer events and hovering inside its body to natively interact with embedded links or elements.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>Boolean</td>
		<td>false</td>
	</tr>
</table>

```jsx
<Tooltip interactive={true} />
```

**`pointerPosition`**

Controls where the pointer appears along the active side (`start`, `center`, `end`).

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>'start' | 'center' | 'end'</td>
		<td>'center'</td>
	</tr>
</table>

```jsx
<Tooltip pointerPosition='start' />
```

**`showPointer`**

Toggles the rendering of the small directional pointer arrow pointing back to the target element.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>Boolean</td>
		<td>true</td>
	</tr>
</table>

```jsx
<Tooltip showPointer={false} />
```

**`className`**

Key to change style through external className.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>String</td>
		<td>''</td>
	</tr>
</table>

```jsx
<Tooltip className='custom-tooltip' />
```
