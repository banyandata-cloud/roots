# IconButton

A square icon-only button with an envelope icon. Available in 6 variants and 4 sizes.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/v2/buttons/iconButton)

> The IconButton is a compact square button that renders an envelope icon. It supports all standard button variants and sizes.

## Usage

```js
import { IconButton } from '@banyan_cloud/roots';
```

- This IconButton component provides a square icon-only interactive element with configurable variants and sizes.

## API

**`variant`**

For applying visual style to the button.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>'primary', 'secondary', 'soft', 'outline', 'ghost', 'critical'</td>
		<td>'primary'</td>
	</tr>
</table>

```jsx
<IconButton variant='secondary' />
```

**`size`**

The size of the button (square dimensions).

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>'xs', 'sm', 'md', 'lg'</td>
		<td>'sm'</td>
	</tr>
</table>

| Size | Button  | Icon    |
| ---- | ------- | ------- |
| xs   | 32×32px | 18×18px |
| sm   | 40×40px | 18×18px |
| md   | 44×44px | 18×18px |
| lg   | 52×52px | 20×20px |

```jsx
<IconButton size='md' />
```

**`disabled`**

If true, the button is disabled and cannot be interacted with.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>Boolean</td>
		<td>false</td>
	</tr>
</table>

```jsx
<IconButton disabled />
```

**`type`**

The type of the button.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>'button', 'submit', 'reset'</td>
		<td>'button'</td>
	</tr>
</table>

```jsx
<IconButton type='button' />
```

**`onClick`**

Callback function to be executed when the button is clicked.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>Function</td>
		<td>{`() => {}`}</td>
	</tr>
</table>

```jsx
<IconButton
	onClick={() => {
		console.log('clicked');
	}}
/>
```

**`blurOnClick`**

If true, the button will lose focus after being clicked.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>Boolean</td>
		<td>true</td>
	</tr>
</table>

```jsx
<IconButton blurOnClick={false} />
```
