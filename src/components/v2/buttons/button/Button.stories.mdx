# Button

A versatile and customizable button component designed for enabling interactive user experiences with configurable appearance and behavior.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library?type=design&node-id=1-2&mode=design&t=doFoJhh78NiStend-0) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/buttons/button)

> The Button component builds upon the BaseButton component, adding more flexibility and visual customization. Pass `leftComponent` and `rightComponent` to render icons inside the button. When only `leftComponent` is passed with no `title`, the button renders as a square icon-only button.

## Usage

```js
import { Button } from '@banyan_cloud/roots';
```

- This Button component enhances the capabilities of the BaseButton and can be effectively used across the application's UI to provide interactive elements with varying styles and behaviors.

## API

### className

Additional CSS class names to be applied to the button.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>String</td>
		<td>""</td>
	</tr>
</table>

```jsx
<Button className='additionalClass' />
```

### title

The text to be displayed within the button. When omitted with only `leftComponent` provided, renders as a square icon-only button.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>string</td>
		<td>""</td>
	</tr>
</table>

```jsx
<Button title='Submit' />
```

### leftComponent

Component to render as the left icon. Receives `className` for sizing. When used alone without `title`, renders the button as icon-only.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>ComponentType</td>
		<td>undefined</td>
	</tr>
</table>

```jsx
<Button leftComponent={MailIcon} />
```

### rightComponent

Component to render as the right icon. Receives `className` for sizing.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>ComponentType</td>
		<td>undefined</td>
	</tr>
</table>

```jsx
<Button rightComponent={ArrowIcon} />
```

### variant

For applying visual style to the button.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>'primary', 'secondary', 'soft', 'outline', 'ghost', 'critical'</td>
		<td>'primary'</td>
	</tr>
</table>

```jsx
<Button variant='secondary' />
```

### size

The size of the button, affecting its dimensions.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>'xs', 'sm', 'md', 'lg'</td>
		<td>'sm'</td>
	</tr>
</table>

```jsx

```

		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>Boolean</td>
		<td>false</td>
	</tr>
</table>

```jsx

```

The type of the button.

		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>


### onClick

Callback function to be executed when the button is clicked.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>Function</td>
		<td>{`() => {}`}</td>
	</tr>
</table>

```jsx

	onClick={() => {
		console.log('clicked');
	}}
/>
```

	<tr>
		<td>Boolean</td>
		<td>true</td>
	</tr>
</table>

