# Skeleton

A placeholder loading component used to indicate that content is being loaded.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/skeleton)

> The Skeleton component provides a visual placeholder while content is loading, improving perceived performance and user experience.

## Usage

```js
import { Skeleton } from '@banyan_cloud/roots';
```
