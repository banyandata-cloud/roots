# Tag

A lightweight label component used to categorize or highlight content. Supports three sizes — `sm`, `md`, and `lg`.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/v2/tag)

## Usage

```js
import { Tag } from '@banyan_cloud/roots';
```

### Variant priority order

When multiple props are combined, the component resolves to one variant in this order:

1. `checkboxIndicatorClosable` → Checkbox-indicator-closable variant (checkbox | dot | label | ×)
2. `checkboxIndicator` → Checkbox-indicator variant (checkbox | dot | label)
3. `indicator + count` → Indicator-count variant (dot | label | badge)
4. `indicator + closable` → Indicator-closable variant (dot | label | ×)
5. `indicator` → Indicator-only variant (dot | label)
6. `checkbox + closable` → Checkbox-closable variant
7. `closable` → Closable variant
8. `checkbox` → Checkbox variant
9. _(none)_ → Default variant
