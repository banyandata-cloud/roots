# Widget

Advanced Component Wrapper with interactive and selective features..

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library?type=design&node-id=1-5&mode=design&t=tXbQWENNIzmFNTVU-0) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/widget)

## Usage

Basic import and additional information on how to use this component.

```js
import { BaseWidget } from '@banyan_cloud/roots';
```
