# Table Cell

Just extends the Base Cell with the property of text overflow ellipsis.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library?type=design&node-id=1-5&mode=design&t=tXbQWENNIzmFNTVU-0) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/table/cell)

> This components uses combination of all Table Elements i.e Cell, Header, Coloum and Row.

## Usage

Basic import and additional information on how to use this component.

```js
import { TableCell } from '@banyan_cloud/roots';
```

## API

**`component1`**

Prop to display Element at left of Base Cell.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>element</td>
		<td>null</td>
	</tr>
</table>

```jsx
<TableCell component1={1} />
```

**`component2`**

Prop to display Element at Center of Base Cell.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>element</td>
		<td>null</td>
	</tr>
</table>

```jsx
<TableCell component2={2} />
```

**`component3`**

Prop to display Element at Right of Base Cell.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>element</td>
		<td>null</td>
	</tr>
</table>

```jsx
<TableCell component3={3} />
```

**`className`**

Key to change style through external className.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>String</td>
		<td>''</td>
	</tr>
</table>

```jsx
<BaseTable className='chart-area' />
```
