# Step

The Step component is used within the Stepper component to represent an individual step within a linear sequence. It can display a title, description, icons indicating completion or error, and optional custom render functions for more advanced customization.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library?node-id=266%3A48405&mode=dev) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/stepper/step)

> The Step component is essential for building a visually engaging and user-friendly stepper UI for multi-step processes, forms, and other linear sequences.

## Usage

Basic import and additional information on how to use this component.

```js
import { Step } from '@banyan_cloud/roots';
```

-   The Step component is used within the Stepper component to display each individual step.
-   The Step component can display different icons based on completion and error states.
-   Custom rendering functions can be used to fully customize the appearance of each step.
-   The Step component is part of the Stepper component and provides flexibility for step customization.
