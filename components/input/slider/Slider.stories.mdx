# Slider

The Slider component provides a customizable Slider variations, including range and normal sliders. It supports controlled and uncontrolled usage and offers various customization options.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library?node-id=151%3A18731&mode=dev) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/input/Slider)

> The Slider component provides an accessible and easily customizable way to implement Slider in your application. It supports both controlled and uncontrolled usage and offers options for variations, range and the Slider Step.

## Usage

Basic import and additional information on how to use this component.

```js
import { Slider } from '@banyan_cloud/roots';
```

-   The Slider component is designed to serve as a customizable input for single-selection options, allowing users to slide and select a value within a specified range.
-   It supports both controlled and uncontrolled usage. When used in a controlled manner (with a defined value and onChange prop), it reflects the provided value. In uncontrolled mode, it manages its own state internally.
-   The component includes a tooltip that dynamically displays the current value of the slider. The tooltip is triggered by hovering over the slider, providing users with visual feedback on the selected value.
-   The component offers support for a range slider, indicated by the range prop. When set to true, it renders a RangeSlider component instead, allowing users to select a range of values.
