# Radio

The Radio component provides a customizable radio input for single-selection options. It supports controlled and uncontrolled usage and offers various customization options.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library?node-id=151%3A18731&mode=dev) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/input/radio)

> The Radio component provides an accessible and easily customizable way to implement radio inputs in your application. It supports both controlled and uncontrolled usage and offers options for positioning and sizing the radio icon.

## Usage

Basic import and additional information on how to use this component.

```js
import { Radio } from '@banyan_cloud/roots';
```

-   The onChange function is called when the radio input state changes. It provides the new checked value as a parameter.
-   The position prop determines the position of the radio input relative to the label.
-   The size prop controls the size of the radio icon.
