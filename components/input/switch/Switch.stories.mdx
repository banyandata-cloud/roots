# Switch

The Switch component provides a customizable toggle switch input. It supports controlled and uncontrolled usage and offers various customization options.

#### Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/input/switch)

> The Switch component provides a sleek and accessible way to implement toggle switches in your application. It supports both controlled and uncontrolled usage and offers options for positioning the switch input.

## Usage

Basic import and additional information on how to use this component.

```js
import { Switch } from '@banyan_cloud/roots';
```

-   The onChange function is called when the switch input state changes. It provides the new checked value as a parameter.
-   The position prop determines the position of the switch input relative to the label.
