export const Template = (args) => {
	const [selectedTab, setSelectedTab] = useState('');
	return (
		<ThemedContainer {...args}>

				}}>
				You have selected {selectedTab}
			</span>
		</ThemedContainer>
	);
};

# Tabs

The Tabs component provides a way to create a tabbed interface with customizable tabs. Each tab can have an associated title, icons, and functionality to manage content switching.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library?node-id=943%3A47943&mode=dev) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/tabs)

> The Tabs component simplifies the creation of tabbed interfaces by providing a customizable way to define tabs and manage their selection. It enhances the user experience by allowing content to be organized and displayed in a structured manner.

## Usage

Basic import and additional information on how to use this component.

```js
import { Tabs } from '@banyan_cloud/roots';
```

-   The tabs prop should be an array of tab objects, where each tab object contains properties like id, title, leftIcon, rightIcon, and disabled.
-   The setSelectedTab function allows you to change the selected tab when a tab is clicked.

## API

**`className`**

Additional CSS class names to be applied to the Tabs container.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>String</td>
		<td>""</td>
	</tr>
</table>

```jsx
<Tabs className='additionalClass' />
```

**`tabs`**

An array of tab objects, where each object represents a tab with properties such as id, title, leftIcon, rightIcon, and disabled.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>Array of string</td>
		<td>[]</td>
	</tr>
</table>

```jsx
<Tabs
	tabs={[
		{
			id: '1',
			title: 'Tab1',
			disabled: true,
		},
	]}
/>
```
