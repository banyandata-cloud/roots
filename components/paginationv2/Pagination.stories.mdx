# Pagination

A versatile pagination component designed to display and navigate through paginated data in React applications.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library?type=design&node-id=1712-44479&mode=design&t=0c2K2ujwYT3NGWlW-0) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/pagination)

> The Pagination component provides navigation controls for paginated data. It accepts a paginationState object to manage the pagination state and a paginationDispatch function for pagination actions.

## Usage

Basic import and additional information on how to use this component.

```js
import { Pagination } from '@banyan_cloud/roots';
```

-   This Pagination component enhances the user experience when dealing with paginated data, providing intuitive controls to navigate through different pages.
-   It's a valuable tool for improving the usability of data-heavy interfaces within your React application.

### Simple Pagination

The default Pagination component.

<Canvas>
	<Story name='Default Pagination'>
		{() => {
			const [paginationState, paginationDispatch] = usePagination({
				totalPages: 40000,
				currentPage: 100,
				totalData: 800000,
			});
			const dataLabel = 'items';
			return (
				<ThemedContainer theme='light'>
					<Pagination
						loading={false}
						dataLabel={dataLabel}
						{...{
							paginationState,
							paginationDispatch,
						}}
					/>
				</ThemedContainer>
			);
		}}
	</Story>
</Canvas>

## API

**`className`**

Additional CSS class names to be applied to the pagination component.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>

	</tr>

		<td>String</td>
		<td>""</td>
	</tr>
</table>

```jsx
<Pagination className='additionalClass' />
```

**`floating`**

If true, the pagination component will float.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>Boolean</td>
		<td>false</td>
	</tr>
</table>

```jsx
<Pagination floating={false} />
```

**`paginationState`**

Pagination state with properties totalPages, currentPage, and step.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>{`{ totalPages: Number, currentPage: Number, step: Number }`}</td>
		<td>{`{ totalPages: null, currentPage: null, step: null }`}</td>
	</tr>
</table>

```jsx
<Pagination paginationState={{ totalPages: 10, currentPage: 1, step: 5 }} />
```

**`paginationDispatch`**

Dispatch function for pagination actions.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>Function</td>
		<td>{`() => {}`}</td>
	</tr>
</table>

```jsx
<Pagination
	paginationDispatch={() => {
		console.log('Something happens');
	}}
/>
```

**`onChange`**

Callback function triggered when the page changes.
