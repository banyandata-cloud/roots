## API

**`variant`**

One of `h1`,`h2`,`b1`,`b2` and `b3`.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>string</td>
		<td>'b2'</td>
	</tr>
</table>

```jsx
<Text variant='b2' />
```

**`component`**

One of `h1`,`h2`,`h3`, `h4`, `h5`, `h6`, `p`,`span` and `a`.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>string</td>
		<td>'span'</td>
	</tr>
</table>

```jsx
<Text component='span' />
```

**`weight`**

One of `100`,`200`,`300`, `400`, `500`, and `600`.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>Number</td>
		<td>'null'</td>
	</tr>
</table>

```jsx
<Text weight={400} />
```

**`stroke`**

One of `regular`,`medium`,`semibold`, and `bold`.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>String</td>
		<td>'regular'</td>
	</tr>
</table>

```jsx
<Text stroke='regular' />
```

**`attrs`**

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>

Key to change style through external className.

	<tr>

		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>String</td>
		<td>''</td>
	</tr>
</table>

```jsx

```
