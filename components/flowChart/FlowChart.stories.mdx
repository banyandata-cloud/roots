# FlowChart

A flowchart component that visualizes data from Neo4j.

#### Design - [Figma](https://www.figma.com) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/FlowChart)

> The FlowChart component provides an easy and customizable way to incorporate flowcharts into your application.By connecting to the Neo4j DataBase and retrieving the data to display in the chart.

## Usage

Basic import and additional information on how to use this component.

```js
import { FlowChart } from '@banyan_cloud/roots';
```

-   The url,user and password are the basic props that let you to connect to the Neo4j database.
-   The query prop allows you to query data from the Neo4j database and display only the information that is required.
-   The other props are used to personalise the node, linkages and fonts in the graph.
