# Area Chart

A Visual Representation of Time Based Data to display value-to-value comparison during a time period through lines or value-to-time comparison through shaded area.

#### Design - [Figma](https://www.figma.com/file/Cx1dHCv47N2i35IP8wNz1u/Charts?type=design&node-id=30-34177&mode=design&t=ZtOwdHlsq1S664aU-0) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/charts/areaChart)

> This components uses the `echarts` utility library to use the Basic Visual Components and properties to customise the same for required use case and `EChartsReactCore` utility library to Wrap the Charts for React Projects.

## Usage

Basic import and additional information on how to use this component.

```js
import { BaseAreaChart } from '@banyan_cloud/roots';
```

-   The Area Chart is just like any other chart with standard `chartData` and `metaData` values alongwith props to control customization.
-   The `className` of the Area Chart can be adjusted in the implementaion code to match its flexibility with respect to its container.

**`seriesData`**

The Actual Data which will be displayed in the Chart. It must have 2 keys - 'chartData' and 'metaData'.
The chartData can have any number of keys representing Data in numerical Array.
The metaData can have any other required Data but must contain 'xAxisData' key which is an array representing Time Series in unix-date format.
