# Pagination

A button component used for user interactions. It provides a versatile interface with customizable features such as size, appearance, and behavior.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library?type=design&node-id=1-2&mode=design&t=doFoJhh78NiStend-0) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/buttons/Pagination)

> The Pagination component renders a button element, encapsulating its behavior and appearance. It is customizable through various props, such as size, disabled, and variant. The onClick prop allows you to define a function to be executed when the button is clicked.

## Usage

Basic import and additional information on how to use this component.

```js
import { Pagination } from '@banyan_cloud/roots';
```

-   This button component can be used in different parts of the application where user interactions are required, providing consistent and adaptable visual elements for your UI.
-   The Pagination component also supports the inheritance of props from the BaseCell component.
