# Side Panel

Component that displays a Container in Side Drawer or Panel.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library?node-id=2%3A157&t=QUwVGAtrtHseOQKo-0) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/sidePanel)

## Usage

Basic import and additional information on how to use this component.

```js
import { BasePanel } from '@banyan_cloud/roots';
```

-   Base Panel uses reference to Display Side Drawer over current DOM.
-   The `className` of the Chart can be adjusted in the implementaion code to match its flexibility with respect to its container.

			const [open, setOpen] = useState(true);
			const toggle = () => {

						renderHeader={
							<div
								style={{
									background: 'red',
								}}>

							</div>
						}
						renderFooter={
							<div
								style={{
									background: 'orange',

						open={open}>
						<div
							style={{
								background: 'tomato',

							}}>
							<p>Body</p>
						</div>
					</BaseSidePanel>
				</div>
			);
		}}
	</Story>
</Canvas>

## API

**`renderHeader`**

Header Element of the Side Panel

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>element</td>
		<td>null</td>
	</tr>
</table>

```jsx
<BaseSidePanel renderHeader={false} />
```

**`renderFooter`**

Footer Element of the Side Panel

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>element</td>
		<td>null</td>
	</tr>
</table>

```jsx
<BaseSidePanel renderFooter={false} />
```

**`open`**

State to decide whether to Open or Close the Side Panel.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>bool</td>
		<td>false</td>
	</tr>
</table>

```jsx
<BaseSidePanel open={true} />
```

**`className`**

Key to change style through external className.

	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
