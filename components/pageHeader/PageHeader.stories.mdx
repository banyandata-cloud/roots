# PageHeader

The PageHeader component is a versatile header component designed for page headers or section headings. It supports customization of the title, description, breadcrumbs, right action, and other elements to provide a comprehensive header for your content.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library?node-id=1%3A8&mode=dev) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/pageHeader)

> The PageHeader component is a flexible solution for creating consistent and appealing headers for different sections or pages within your application. It combines various elements like title, description, breadcrumbs, and custom actions to enhance the presentation of your content.

## Usage

Basic import and additional information on how to use this component.

```js
import { PageHeader } from '@banyan_cloud/roots';
```

-   The theme prop allows you to choose between a dark or light theme for the header.
-   The crumbsProps prop allows you to configure the breadcrumbs using the same props as the BreadCrumbs component.
-   The rightAction prop accepts a custom component that will be rendered as a right action in the header.

The main title of the header.
