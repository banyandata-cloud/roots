# Hierarchy Item

A versatile component for rendering hierarchical items within a navigation or browser-like interface in React applications. This component provides collapsible sections with expand/collapse functionality.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library?type=design&node-id=943-47941&mode=design&t=XnPIAzJOOq0Pc2Yi-0) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/hierarchy/item)

> The HierarchyItem component renders hierarchical items with expand/collapse functionality and a clickable title. It can contain children nodes, forming a nested hierarchy.

## Usage

Basic import and additional information on how to use this component.

```js
import { HierarchyItem } from '@banyan_cloud/roots';
```

-   This HierarchyItem component enhances the readability and organization of hierarchical data, allowing users to expand or collapse sections to navigate through nested content. It's a useful building block for creating navigational components in your React application.

## API

**`iconPlacement`**

The placement of the expand/collapse icon.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>'left', 'right', 'none'</td>
		<td>'left'</td>
	</tr>
</table>

```jsx
<HierarchyItem iconPlacement='right' />
```

**`title`**

The content to be displayed as the title of the hierarchy item.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>JSX</td>
		<td>null</td>
	</tr>
</table>

```jsx
<HierarchyItem title={<p>Title</p>} />
```

**`defaultOpen`**

If true, the item is initially open (expanded).

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>Boolean</td>
		<td>false</td>
	</tr>
</table>

```jsx
<HierarchyItem iconPlacement='right' />
```

**`onClick`**

Callback function triggered when the item's title area is clicked.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>Function</td>
		<td>{`() => {}`}</td>
	</tr>
</table>

```jsx
<HierarchyItem
	onClick={() => {
		console.log('Button clicked');
	}}
/>
```

**`active`**

If true, the item is marked as active.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>
		<th style={{ textAlign: 'left', width: '50%' }}>DEFAULT</th>
	</tr>
	<tr>
		<td>Boolean</td>
		<td>false</td>
	</tr>
</table>

```jsx
<HierarchyItem active={true} />
```
