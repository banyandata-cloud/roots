# Chip

A versatile chip component designed to represent small blocks of information or actions. It utilizes the BaseButton component to provide customizable and interactive chip elements.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library?type=design&node-id=135-18700&mode=design&t=COZ0piydDEh1Sggi-0) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/buttons/chip)

> The Chip component builds upon the BaseButton component, providing a flexible and customizable way to represent various pieces of content or actions as chips. You can include custom components (leftComponent and rightComponent) to enhance the chip's content.

## Usage

Basic import and additional information on how to use this component.

```js
import { Chip } from '@banyan_cloud/roots';
```

-   This Chip component adapts the capabilities of the BaseButton and can be effectively used to represent data or actions in a compact and interactive format within your application's UI.
