# Button

A versatile and customizable button component designed for enabling interactive user experiences with configurable appearance and behavior.

#### Design - [Figma](https://www.figma.com/file/e9opoAtQHBo4vFd6u0Co98/Component-Library?type=design&node-id=1-2&mode=design&t=doFoJhh78NiStend-0) | Code - [Github](https://github.com/banyandata-cloud/roots/tree/main/components/buttons/Button)

> This components uses the `date-fns` utility library to use date related methods with respect to comparision and sorting.

## Usage

Basic import and additional information on how to use this component.

```js
import { Button } from '@banyan_cloud/roots';
```

-   Treat Date Picker as another dropdown(select) element in terms of HTML, as both uses the same `<PopOver` Wrapper.
-   The `width` of the date picker has to adjusted in the implementaion code to match its flexibility with respect to its container.

## API

**`className`**

Additional CSS class names to be applied to the button.

<table style={{ width: '100%' }}>
	<tr>
		<th style={{ textAlign: 'left', width: '50%' }}>TYPE</th>

	</tr>
	<tr>
		<td>String</td>
		<td>""</td>
	</tr>
</table>

```jsx
<Button className='additionalClass' />
```

**`component1`**

The first component on the left of the title. Accepts JSX element.
